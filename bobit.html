Bobit

What is Bobit?

Bobit is a simple discord bot i coded in rust using serenity.

How do I invite Bobit to my server?

Bobit is currently not avaliable to add to servers, look below for an alternative.

How do I host Bobit myself?

Grab the artifact from the latest Action, unzip the file and create a file named bobit.json then use this format in it
{
"token": (bot token here in ""),
"prefix": (your choice of the prefix also in "")
}
Then just run the bobit.exe
